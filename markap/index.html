Селектори,Специфіка:

  • *{margin: 10px;} - універсальний селектор, задасть відступи в 10px для всіх елементів сайту.

  • h1 {color: blue;} - теговий селектор, задасть синій колір для тексту тегу h1.

  • .class1 {color: red;} - селектор классу, задасть червоний колір для всіх елементів з ім'ям классу class1.

  • #zf {color: green;} - селектор ідентифікатора, задасть зелений колір для елементу з унікальним ім'ям ідентифіктора zf.

  • h3, h2, a {backgraund: red;} - селектор групп, задасть червоний колір фону всіх елементів блоку h3,h2 та a
